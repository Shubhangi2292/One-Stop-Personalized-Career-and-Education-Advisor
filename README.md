# 🎯 One Stop Personalized Career and Education Advisor

An AI-powered web-based platform designed to help students and job seekers receive personalized career recommendations, educational guidance, resume building support, and progress tracking through an interactive dashboard.

---

## 📌 Project Overview

The **One Stop Personalized Career and Education Advisor** is an intelligent recommendation system that provides users with suitable career paths based on their academic stream, interests, and preferences.

The platform offers:
- 🎓 Personalized career recommendations
- 📚 Educational guidance
- 📄 Resume builder
- 📊 Admin analytics dashboard
- 👥 User activity tracking
- 🗂 Career database editor

This project aims to simplify career decision-making using AI-driven logic and data analytics.

---

## 🚀 Features

### 👤 User Module
- User registration and login
- Career recommendation based on stream and interests
- 7-day trial progress tracking
- Resume generation
- Career exploration

### 🛡 Admin Module
- Total users analytics
- Active and completed trial count
- Top selected careers graph
- Stream-wise user statistics
- User management and deletion
- Activity logs
- Career CSV editor
